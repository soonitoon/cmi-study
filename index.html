1주차 과제: 캘린더 만들기


  • 클릭!

  • 요구사항

    • HTML과 CSS를 사용하여 간단한 캘린더 만들기(JS를 사용한 기능 구현도 가능)📆
    • 만드시 CSS flex를 사용할 것.
    • 개인 GitHub 리포지토리에 과제 올리기.

  • 결과물

    • 남색 톤의 심플한 디자인으로 구현.🎨
    • flex를 사용해 캘린더 중앙정렬.
    • grid를 사용해 캘린더 템플릿 제작.
    • JS의 Date 인스턴스를 사용해 각 년월에 맞는 날짜 템플릿 로드.
